# 곡선-장르/템포 매핑 (휴리스틱)

이 문서는 표준 규칙이 아니라 **휴리스틱**이다.
- 곡선 선택은 **세션 길이, 콘텐츠 티어 수, 보상 구조**에 따라 달라진다.
- 아래 예시는 **곡선 형태가 체감 속도에 미치는 영향**을 기준으로 한 추정이다.
- 팀 합의가 있으면 이 표를 업데이트해도 된다.

## 진행/경험치 (Progression)
| 모델 패밀리 | 템포 적합 | 장르 예시 | 선택 이유 |
|---|---|---|---|
| Linear / 낮은 Power (p≈1~1.4) | 빠른 템포 | 세션형 액션, 아케이드, 짧은 런의 로그라이크 | 레벨업 간격이 크게 벌어지지 않아 리듬이 유지됨 |
| 중간 Power (p≈1.6~2.2) | 중간 템포 | ARPG, JRPG, 오픈월드 RPG | 초반은 빠르고 후반은 완만하게 느려지는 체감 설계에 유리 |
| Exponential/Geometric | 느린 템포/장기 | MMORPG, 아이들/인크리멘탈 | 후반 요구량이 커져 장기 목표 설계에 적합 |
| Sigmoid/Logistic | 빠른 온보딩 → 안정화 | 캐주얼 RPG, 수집형/가챠 | 초반 진입 허들을 낮추고 중반을 안정화하기 좋음 |
| Piecewise | 혼합 템포 | 챕터/시즌형, 캠페인 기반 | 구간별 체감 변화를 의도적으로 만들기 쉬움 |

## 전투 난이도 (Difficulty/EHP)
| 모델 패밀리 | 템포 적합 | 장르 예시 | 선택 이유 |
|---|---|---|---|
| Linear EHP | 일정 템포 | 전술/턴제, 안정적 난이도 상승 | 실력 상승과 맞물려 예측 가능한 난이도 제공 |
| Exponential EHP | 후반 고강도 | 하드코어 RPG, 엔드게임 집중형 | 후반 챌린지를 강조할 때 유리 |
| Piecewise | 구간 강조 | 보스/챕터 중심 구조 | 특정 구간에서 난이도 전환을 명확히 하기 쉬움 |

## 드랍/경제 (Drops/Economy)
| 모델 패밀리 | 템포 적합 | 장르 예시 | 선택 이유 |
|---|---|---|---|
| Diminishing Returns | 안정화 | 장기 운영, 과성장 방지 필요 | 상위 구간 과잉 성장을 억제 |
| Pity/Soft Cap | 좌절 완화 | 가챠/강화 기반 | 실패 누적 좌절을 줄여 공정감 강화 |
| Geometric | 희귀성 강조 | 수집형/파밍 중심 | 희귀 아이템 추적 동기 유지 |

## 빠른 선택 질문
- 세션 길이는? (10~20분 / 30~60분 / 2시간+)
- 주요 체감 목표는? (빠른 성장 / 완만한 성장 / 후반 장기)
- 콘텐츠 티어 수는? (적을수록 Linear/낮은 Power가 유리)
