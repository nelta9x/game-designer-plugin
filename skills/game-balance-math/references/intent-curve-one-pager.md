# 체감 의도-곡선 1페이지 요약

이 문서는 상담 시 바로 제시하는 빠른 추천표다.  
상세 설계 근거와 사례는 `growth-intent-curve-playbook.md`를 참조한다.

## 빠른 추천표

| 사용자 의도 | 1순위 | 2순위 | 피해야 할 신호 | 첫 조정 레버 | 필수 검증 지표 |
|---|---|---|---|---|---|
| 예측 가능하고 공정한 성장 | 선형(Linear) | 낮은 파워(p≈1.1~1.4) | 후반 목표가 너무 빨리 소진됨 | base, 레벨당 증가량 | 레벨업 시간, 이탈률 |
| 초반 도파민 + 후반 장기 목표 | 지수(Exponential) | 중간 파워(p≈1.6~2.2) | 중반 급벽(급격한 ratio 증가) | ratio, 구간별 ratio | 레벨업 시간 p90, 잔존율 |
| 챕터 전환에서 긴장감 | 계단식(Piecewise) | 선형+구간 배율 | 경계 점프(29->30 급등) | 경계 위치, 경계 완충 2~3레벨 | 경계 전후 이탈률, 레벨업 시간 |
| 온보딩 빠르고 중반 안정화 | 로지스틱(Sigmoid) | 낮은 파워+소프트캡 | 상한이 너무 이르게 도달 | k, x0, 상/하한 | 구간별 레벨업 시간, 첫 시도 클리어율 |
| 상위권 과성장 억제 | 체감(Diminishing) | 로지스틱 캡 | 투자 가치 급락 체감 | k(반감점), 상한값 | 상위권 성장률, 경제 TTE |
| 단조로움 해소(중후반 자극) | 선형 -> 계단식 전환 | 선형 -> 파워 전환 | 전 구간 선형 고정 | 전환 구간, 증가 기울기 | 구간별 체감 설문, 세션 길이 |
| 벽 체감 완화(막힘 제거) | 계단식 완충 | 지수 -> 파워 완화 | 경계 기울기 급변 | 경계 완충, ratio 하향 | 경계 구간 p90, 재시도 횟수 |
| 계산 가능성을 유지한 난이도 상승 | 선형+낮은 파워 | 계단식 소폭 전환 | 수식/규칙 설명 불가 | 파워 지수 p, 구간별 규칙 | 예측 오차, 유저 불신 피드백 |

## 상담 답변 체크리스트

- 의도 요약 1줄
- 모델 2개 이상 제안(우선순위)
- 왜 이 모델인지(체감/시스템 근거)
- 재미 가설 1줄(변경 -> 기대 행동)
- 리스크 1~2개
- 조정 레버 1~2개
- 검증 지표 2~3개
