# 수확 체감 (Diminishing Returns) 예시

## 빠른 내비게이션

### 언제 이 문서를 여는가

- 특정 스탯이 후반에 과도하게 강해지거나, 반대로 투자 체감이 사라질 때
- 쿨감/공속/방어/치명/이속 같은 비율형 스탯의 상한을 부드럽게 제어할 때
- k값(50% 도달점)을 장르 템포에 맞춰 튜닝해야 할 때

### Quick Formula

```
감소율 = x / (x + k)
배율 = 1 + x / (x + k)
피해감소 = 방어력 / (방어력 + 상수)
치명타율(시그모이드) = 최대율 / (1 + e^(-k*(x-x0)))
```

### 목차

- 1. 쿨다운 감소
- 2. 공격 속도
- 3. 방어력 -> 피해 감소
- 4. 치명타 확률
- 5. 이동 속도
- 6. 경험치 보너스
- k값 설계 가이드
- 설계 체크리스트

## 1. 쿨다운 감소

### 공식: `감소율 = x / (x + k)`

k = 100일 때:

| 쿨감 스탯 | 감소율 | 기본 10초 쿨 → |
|-----------|--------|----------------|
| 0 | 0% | 10.0초 |
| 50 | 33% | 6.7초 |
| 100 | 50% | 5.0초 |
| 200 | 67% | 3.3초 |
| 300 | 75% | 2.5초 |
| 500 | 83% | 1.7초 |
| 1000 | 91% | 0.9초 |

**특징**: 0초에 절대 도달 불가

### k값에 따른 차이

| k값 | 50% 도달점 | 75% 도달점 | 용도 |
|-----|-----------|-----------|------|
| 50 | 50 | 150 | 빠른 체감 |
| 100 | 100 | 300 | 중간 |
| 200 | 200 | 600 | 느린 체감 |

---

## 2. 공격 속도

### 공식: `공속 배율 = 1 + x / (x + k)`

k = 100일 때:

| 공속 스탯 | 배율 | 기본 1초 → |
|-----------|------|-----------|
| 0 | 1.00x | 1.00초 |
| 50 | 1.33x | 0.75초 |
| 100 | 1.50x | 0.67초 |
| 200 | 1.67x | 0.60초 |
| 500 | 1.83x | 0.55초 |

**최대**: 2.0배 (0.5초)에 점근

---

## 3. 방어력 → 피해 감소

### 공식: `피해감소 = 방어력 / (방어력 + 상수)`

상수 = 100 + (공격자레벨 × 10)

레벨 10 공격자 (상수 = 200):

| 방어력 | 피해 감소 |
|--------|----------|
| 0 | 0% |
| 100 | 33% |
| 200 | 50% |
| 400 | 67% |
| 800 | 80% |

**특징**: 레벨이 높은 적에게는 방어력 효율 감소

---

## 4. 치명타 확률

### 문제: 100% 초과 방지

나쁜 예: `치명타율 = 기민성 × 0.5%`
→ 기민성 200이면 100%? 250이면 125%?

### 좋은 예: 시그모이드

```
치명타율 = 최대율 / (1 + e^(-k × (기민성 - 중간점)))
```

최대율=75%, k=0.05, 중간점=100:

| 기민성 | 치명타율 |
|--------|---------|
| 0 | 0.5% |
| 50 | 8% |
| 100 | 37.5% |
| 150 | 67% |
| 200 | 74% |
| 300 | 75% |

**특징**: 자연스럽게 75%에 수렴

---

## 5. 이동 속도

### 공식: `최종속도 = 기본 × (1 + 보너스 / (보너스 + k))`

k = 50, 기본속도 = 5:

| 이속 보너스 | 배율 | 최종 속도 |
|------------|------|----------|
| 0 | 1.00x | 5.0 |
| 25 | 1.33x | 6.7 |
| 50 | 1.50x | 7.5 |
| 100 | 1.67x | 8.3 |
| 200 | 1.80x | 9.0 |

**최대**: 2배 (10.0)에 점근

---

## 6. 경험치 보너스

### 선형 (위험)

```
경험치 배율 = 1 + 스탯 × 0.01
```

스탯 100 = 2배, 스탯 500 = 6배 (밸런스 붕괴)

### 역수 (안전)

```
경험치 배율 = 1 + 스탯 / (스탯 + 100)
```

| 경험치 스탯 | 배율 |
|------------|------|
| 0 | 1.0x |
| 100 | 1.5x |
| 300 | 1.75x |
| 500 | 1.83x |

**최대**: 2배

## k값 설계 가이드

| 게임 스타일 | k값 기준 | 이유 |
|------------|---------|------|
| 캐주얼 | 낮음 (50) | 빠른 보상 체감 |
| 코어 | 중간 (100~200) | 장기 성장 동기 |
| 하드코어 | 높음 (300+) | 극한 최적화 여지 |
